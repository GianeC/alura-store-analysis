# 📊  Análisis de rendimiento de tiendas - Alura Store
## Descripción del proyecto

Este proyecto tiene como objetivo analizar el rendimiento de las cuatro tiendas de la cadena Alura Store para identificar cuál presenta el menor desempeño y recomendar cuál debería vender el Sr. Juan.

El análisis se realizó utilizando Python y la biblioteca Pandas, evaluando diferentes métricas clave del negocio para comparar el desempeño de cada tienda.

## Objetivo del análisis

Determinar cuál tienda tiene el menor rendimiento general considerando diferentes indicadores de negocio como:

* Ingresos totales por tienda
* Categorías de productos más vendidas
* Calificación promedio de los clientes
* Productos más y menos vendidos
* Costo promedio de envío

## Tecnologías utilizadas

* Python
* Pandas
* Matplotlib
*Jupyter Notebook / Google Colab
* GitHub
